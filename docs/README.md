# Documentation index

Project documentation lives in **per-branch folders** under `docs/`:

- **`docs/main/`** — documents the repository as of the **`main`** branch.

When you ship work from another Git branch, copy or author a matching folder (for example `docs/your-feature/`) and link it from your PR description.

Start here: [docs/main/README.md](./main/README.md).
